Theoretical Analysis:

PF permanent regime $\equiv$ Stability of ODE system $$ \begin{split} 0=&\frac{1}{\lambda_1}-\frac{\lambda_1\overline{\theta}_1^2}{\left( \lambda_1\overline{\theta}_1+\lambda_2\overline{\theta}_2\right)^2}-\overline{\theta}_1\\ 0=&\frac{1}{\lambda_2}-\frac{\lambda_2\overline{\theta}_2^2}{\left( \lambda_1\overline{\theta}_1+\lambda_2\overline{\theta}_2\right)^2}-\overline{\theta}_2\\ \end{split} $$
Extension to RNs:
  1. Two types of connections
    1. $n_\mathrm{d}$ direct users
    2. $n_\mathrm{r}$ relay users
  2. Two types of frames:
    1. $\alpha$ all connections allowed
    2. $1-\alpha$ only direct connections
Result: approx. time-sharing
  1. $\alpha\times$ rate of a $n_\mathrm{d}+n_\mathrm{r}$-user PF scheduler
  2. +$(1-\alpha)\times$ rate of a $n_\mathrm{d}$-user PF scheduler
$$ \begin{array}{cccc} 0=\color{#005682}{\alpha\left[\frac{1}{\lambda_{1,r}}-\frac{\lambda_{1,r}\overline{\theta}_1^2}{\left(\lambda_{1,r}\overline{\theta}_1+\lambda_{2,r}\beta\overline{\theta}_2\right)^2}\right]} +\color{#008141}{(1-\alpha)\left[\frac{1}{\lambda_{1,a}}-\overline{\theta}_1\right]}-\overline{\theta}_1\\ 0=\color{#005682}{\alpha\left[\frac{1}{\lambda_{2,r}}-\frac{(\overline{\theta}_2\beta)^2\lambda_{2,r}}{\left( \lambda_{1,r}\overline{\theta}_1+\lambda_{2,r}\beta\overline{\theta}_2\right)^2}\right]}-\overline{\theta}_2\\ \end{array} $$
  1. $\alpha\times$ rate of a $n_\mathrm{d}+n_\mathrm{r}$-user PF scheduler (in all users)
  2. $(1-\alpha)\times$ rate of a $n_\mathrm{d}$-user PF scheduler (in direct users only)
Scheduler Analysis III

Contribution C1.2.8: Scheduling Parameters Trade-offs

Incentive parameter $\beta>1$
  1. Proposed in literature for link balancing
  2. Bias to RN selection
  3. Degrades multi-user diversity
Similar analises conclude that
  1. Interference $\frac{\lambda_{\mathrm{d}}^{(2)}}{\lambda_\mathrm{d}}$ makes direct users worse
  2. Time-split factor $\alpha\to 0$ causes RN starvation
  3. Time-split factor $\alpha\to 1$ causes two-hop link inbalance
  4. Recommendation: increase $\alpha$ and make incentive $\beta<1$ (reverse incentive)

Capacity Scaling Law

Contribution C1.5.1: Throughput Capacity Scaling Laws for Wideband Cellular Networks

The rate from the BS to each node is upper bounded by a function with scaling \begin{equation} R_{\mathrm{DL}(n)}\leq \Theta\left(n^{\beta-1+\min\left(\psi+\gamma,(1-\nu)\frac{\alpha}{2}\right)}\right) \end{equation}

IMH DL feasible rate per node scales as \begin{equation} R_{\mathrm{IMH-DL}}(n)\sim \Theta\left(n^{\beta-1+\min\left(\psi+\gamma,(1-\nu)\frac{\alpha}{2}\right)}\right) \end{equation}

Upper bound layout

  1. Cut set bound to rate
  2. All BS and all nodes pool antennas together in a cooperative massive MIMO
  3. Transition-to-power-limited distantce $r_{\mathrm{v}}=\Theta(n^{-(\gamma+\psi)\frac{2}{\alpha}})$
  4. Three regimes:
    1. $r_{\mathrm{v}}>r_\mathrm{cell}$: all nodes DoF limited
    2. $r_\mathrm{cell}>r_{\mathrm{v}}>n^{(1-\nu)\frac{\alpha}{2}}$: all nodes power-limited at $r_\mathrm{v}$
    3. $n^{(1-\nu)\frac{\alpha}{2}}>r_{\mathrm{v}}$: all nodes power-limited at $n^{(1-\nu)\frac{\alpha}{2}}$
  1. Each point to point link with
    1. Bandwidth $W_\mathrm{u}(n)$,
    2. $\ell_t(n)$ transmission antennas,
    3. $\ell_r(n)$ reception antennas
    4. $\displaystyle \mathrm{SINR}_{u}(n)=\frac{P_\mathrm{u}(n)d(n)^{-\alpha}}{N_\mathrm{I}(n)W_\mathrm{u}(n)}$
    scales with $n$ as \begin{equation} R_{\mathrm{u}}(n) =\begin{cases}\Theta\left( W_{\mathrm{u}}(n)\min(\ell_t(n),\ell_r(n))\right)& W_{\mathrm{u}}^*\geq\Theta\left(W_{\mathrm{u}}(n)\right)\\ \Theta\left(\ell_r(n)\frac{P_\mathrm{u}(n)d(n)^{-\alpha}}{N_\mathrm{I}(n)} \right)& W_{\mathrm{u}}^*<o(W_{\mathrm{u}}(n))\\ \end{cases} \end{equation} where $W_{\mathrm{u}}^*(n):=\frac{\ell_r}{\min(\ell_\mathrm{t},\ell_\mathrm{r})}\frac{P_\mathrm{u}(n)d(n)^{-\alpha}}{N_\mathrm{I}(n)}$
  2. The noise+interference PSD $N_{\mathrm{I}}$ scales as \begin{equation} \label{eq:NIDL} N_{\mathrm{I,IMH-DL}}=\Theta\left(n^{\max\left(\frac{\alpha}{2}(1-\nu)-\psi-\gamma,0\right)}\right) \end{equation}
  3. When $N_{\mathrm{I}}$ is dominated by interference (first term), all nodes lie (W.H.P.) before a critical distance ($r^*$ such that $W_{\mathrm{u}}^*(n)<W_{\mathrm{u}}$), and vice-versa.

  1. $B\to\infty$ and $N_\mathrm{r}\to\infty$, ¿SNR?
  2. $\epsilon:=\lim_{B,N_{\mathrm{r}}\to\infty}\frac{\log B}{\log N_{\mathrm{r}}}$
  3. $B=N_{\mathrm{r}}^{\epsilon}$
  4. Rate limited by $$ R=\begin{cases} \Theta(W)&\epsilon\leq\epsilon_{th}\\ \Theta(\frac{P}{N_0})&\epsilon>\epsilon_{th}\\ \end{cases} $$
  5. Coherent channel trivial
    1. Coherent combining gain $N_{\mathrm{r}}$
    2. $\mathrm{SNR}\propto\frac{N_{\mathrm{r}}}{B}$
    3. $\epsilon<1\Rightarrow \mathrm{SNR}\to \infty$
    4. $\epsilon>1\Rightarrow \mathrm{SNR}\to 0$
Non coherent channel $\epsilon_{th}$
  1. Critical Bandwidth $\propto \frac{P}{N_0}\sqrt{N_\mathrm{r}}$, suggests $\epsilon_{th}=\frac{1}{2}$
  2. But the wideband analysis does not hold when $\epsilon<1$
  3. Alternate upper bound:
    1. Received signal vector $\mathbf{y}\in \mathbb{C}^{1\times N_{\mathrm{r}}}$
    2. $|\mathbf{y}|^2$ sufficient statistic
    3. $\mathrm{E}[|\mathbf{y}|^2]=\frac{P\sqrt{N_{\mathrm{r}}}}{N_0B}$
    4. If $\epsilon>\frac{1}{2}$ and $R>\Theta(\frac{P}{N_0})$, errors
  4. Achievable with multicarrier Pulse Energy Modulation
  5. $\sqrt{N_{\mathrm{r}}}$ diversity gain

Conclusions and Future Work
  1. The point-to-point link model is nearly exhausted
  2. Future wireless networks require cooperative network solutions
  3. Cooperative diversity gain analysis
    1. Relay channel
    2. PHY techniques
    3. Cooperative MAC functions
  4. Current LTE-A standard too conservative
    1. Fixed relaying only
    2. Insufficient understanding of multi-hop
    3. Prioritizes backwards compatibility and simplicity over optimized operation
  1. Short term: cooperation through Cognitive Radio
    1. Stubborn licensees use spectrum inefficiently
    2. Cooperative Spectrum Leasing
    3. Static scheme: only for very bad primaries
    4. Dynamic scheme: minor to high gain depending on efficiency of primary
  2. Potential for CSL in LTE-A
    1. Puny gain with a single RN
    2. Massive RN populations attractive, but still moderate gain
    3. Niche applications: M2M, SG, P2P
  1. Long term: next standard generation
    1. Massive degrees of freedom (spatial, bandwidth, antennas...)
    2. Approaching power limited capacity regimes
    3. Imperative need of cooperative multi-hop
    4. The less cooperative a protocol, the more limited
  2. Re-think wireless networking: cooperative multi-hop by design
    1. Understand the triple interaction:
      • Large bandwidth
      • Multi user allocation
      • Capacity
    2. Implement massive bandwidth signaling
      • Non-peaky and coherent (pilots)
      • Peaky and non-coherent (FSK)
    3. Massive no. of reception antennas
    4. Dynamic Duplex link selection
      • Multi-user tree-topology practical power allocation
      • Single-user mesh-topology capacity-approaching scheduling
